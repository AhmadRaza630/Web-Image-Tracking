  $ python3 generate_markers.py


STEP 2️⃣: START WEB SERVER (1 minute)
──────────────────────────────────────

Open Terminal and run:
  $ cd /Users/ghurki/Desktop/Ahmad\ Ghurki/Web-Image-Tracking
  $ python3 -m http.server 8000

You should see:
  Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...


STEP 3️⃣: OPEN APPLICATION (1 minute)
──────────────────────────────────────

1. Open Web Browser
2. Go to: http://localhost:8000
3. Click on: index.html
4. Grant camera permission when asked
5. Point camera at printed marker image
6. Click "Start Rotation" button
7. Watch your 3D model rotate! 🎉

🆘 QUICK TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════

Issue: "Model not showing"
Fix: Check that .patt files exist in Tracker/ folder

Issue: "Marker not detected"
Fix: Good lighting is key. Print markers on paper, not screen.

Issue: "Camera not working"
Fix: Check browser camera permissions (Settings > Privacy > Camera)

Issue: "Rotation button not working"
Fix: Ensure marker is visible to camera, then click button

More Help: See README.md Troubleshooting section
